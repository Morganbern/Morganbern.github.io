Detection of malicious attack

I got the opportunity to carry an IA and cyber-security project as a student in the Esaip. The aim is to detect malicious attack like DDOS (Distributed denial of service), SQL injection, command injection and HTTP malicious request. Firstly, we defined on what type of malicious we would like ot work on. Next, we looked for databases to make learn the IA and to carry some test. Secondly, after we had sort the databases, we searched IA algorithms to exploit the databases like classfication algorithms.
